# Instalación
## 

    cd server
    npm i
    cd ../multiwireless
    npm i


# Ejecución

## En la primera terminal
    cd server
    npm run start:server
    
## En la segunda terminal
    cd multiwireless
    ng serve

# Las vistas se encuentran las siguientes URL:
    http://localhost:4200/dashboard-analista
    http://localhost:4200/dashboard-jefe-de-flota
